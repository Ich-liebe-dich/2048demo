# 2048demo
一个基于EasyX开发的C/C++版本的2048demo
 - EasyX是C++的一个图形库，主要作用是弥补 turbo C 中的```<graphics.h>```，时界面更加好看 
 - 下载完导入后可能需要去网上下载EasyX库才能运行，地址：[https://easyx.cn/]
